==============================================
== 确保下面的模块
==============================================
php-mysql 必须
php-gd 必须 
php-mbstring 必须  
php-fsockopen 必须
php-json 必须
php-curl  必须


==============================================
== 以下目录设为可写
==============================================
config/config.inc.php   系统配置文件
data/
data/avatar/  用户头象文件
data/data_cache   文件缓存
data/tpl_c/  php运行文件
data/uploads/ 用户上传文件
tpl/default/ 下模板文件需要可写（如果需要在线编辑的话）


==============================================
== 安装脚本
==============================================

通过浏览执行 index.php
管理入口，创始人登录后在下拉菜单中可以看到
安装完后，请删除 install目录
